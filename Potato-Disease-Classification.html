Potato Disease Classification

Project information

  • Category: Deep Learning
  • Project Name: Potato Disease Classification
  • Project Date: 21 August, 2024
  • Project URL: Click Here

Developed a deep learning model to accurately identify potato diseases using TensorFlow and Keras. Trained on the PlantVillage dataset, the model distinguishes between early blight, late blight, and healthy potatoes. Integrated with a local FastAPI web application for demonstration purposes. Future plans include expanding the model to include tomato diseases and enhancing its capabilities.
