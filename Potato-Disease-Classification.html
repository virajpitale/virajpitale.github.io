Potato Disease Classification

Project Information

  • Category: Deep Learning
  • Project Name: Potato Disease Classification System
  • Project Date: 20 July, 2024
  • Project URL: Click Here

This project leverages deep learning to classify potato plant diseases with high accuracy. Using TensorFlow and Keras, the model achieves over 98% accuracy for Early Blight, Late Blight, and Healthy categories. The system features a simple HTML/CSS frontend and a FastAPI server running locally.
